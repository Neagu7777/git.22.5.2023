1. Как создать git репозиторий из локального каталога на своем компьютере. Как его отправить на github.com?
2.Как отказаться от изменений в файле (файлах) находящийся в рабочем каталоге (work directory)?
3.Как вернуть файл из индекса в рабочий каталог?
4.Как отменить ошибочный commit ?

В ответах на вопросы 1-4 необходимо указать конкретные команда git. 

1. Создать git репозиторий из локального каталога:
   
   git init
   git add .
   git commit -m "Initial commit"
   

   Отправить на GitHub:
   - Создайть репозиторий на GitHub.
   - Скопировать URL репозитория.
   - Связать локальный репозиторий с удаленным:
     
     git remote add origin <URL репозитория>
     
   - Отправить изменения:
     
     git push -u origin master
     

2. Отказаться от изменений в файле(ах) в рабочем каталоге:
   
   git checkout -- <название файла>
   

3. Вернуть файл из индекса в рабочий каталог:
   
   git reset HEAD <название файла>
   

4. Отменить ошибочный commit:
   - Сохраните изменения в рабочем каталоге:
     
     git stash
     
   - Отменить последний commit, сохраняя изменения:
     
     git reset --soft HEAD^
     
   - Применить ранее сохраненные изменения:
     
     git stash apply
     
   - Создайть новый commit с корректировками:
     
     git commit -c ORIG_HEAD
     
     Или, если изменения устраивают, можно просто:
     
     git commit --amend
